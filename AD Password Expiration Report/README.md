# AD Password Expiration Report

Dumps the fields below for **all** AD users to C:\ltworking\adpasswords.csv then uploads to the computer specific Uploads folder on the LTShare.

DistinguishedName  
Enabled  
GivenName  
LastLogonDate  
Name  
ObjectClass  
ObjectGUID  
PasswordLastSet  
PasswordNeverExpires  
SamAccountName  
SID  
Surname  
UserPrincipalName
